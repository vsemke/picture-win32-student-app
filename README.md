

Это приложение на C/C++ для Windows. Оно предоставляет графический интерфейс для построения параболы по трём точкам и сохранения результата в файл BMP.

## Возможности
- Построение параболы y = ax² + bx + c по трём кликам мыши.
- Автоматическое вычисление коэффициентов a, b, c.
- Сохранение нарисованного содержимого в файл формата BMP.
- Многоязычный интерфейс (поддержка UTF-8 через lang.ini).
- Рисование крестиков в отмеченных точках.

## Требования
- Windows (системные зависимости: windows.h, commdlg.h).
- Компилятор C/C++, способный компилировать Win32 GUI-приложения.

## Сборка и запуск
1. Откройте проект в своей IDE (например, Visual Studio) или соберите командой:
   ```
   cl main.cpp /link /subsystem:windows /manifestdependency:"type='win32' name='Microsoft.Windows.Common-Controls' version='6.0.0.0' processorArchitecture='*' publicKeyToken='6595b64144ccf1df' language='*'"
   ```
2. Поместите файл `lang.ini` (если нужна локализация) в ту же папку, что и исполняемый файл.
3. Запустите сгенерированный EXE.

## Использование
1. Запустите приложение.
2. По клику в окно приложения укажите три точки — они станут вершинами параболы.
3. После третьего клика парабола будет вычислена и выведена; если точки коллинеарны, показывается ошибка.
4. Чтобы сохранить изображение, выберите в меню Файл → Сохранить как… и укажите путь для BMP.
5. Чтобы начать заново, выберите Файл → Новый или нажмите кнопку создания нового задания.
